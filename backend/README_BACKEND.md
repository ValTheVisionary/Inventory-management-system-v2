# Backend API

## Setup
1. npm install
2. Set DATABASE_URL and JWT_SECRET env vars
3. npx prisma migrate dev
4. npm run dev

## Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET/POST/PUT/DELETE /api/products
- GET /api/categories
- GET /api/dashboard
- POST /api/stock/adjust
- GET /api/users
